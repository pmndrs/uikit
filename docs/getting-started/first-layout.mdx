The first step for building a layout is to install the dependencies.

`npm i three @ni2khanna/uikit @pmndrs/pointer-events`

Next, we create the `index.ts` file and import the necessary dependencies.

```ts showLineNumbers
import { PerspectiveCamera, Scene, WebGLRenderer } from 'three'
import { reversePainterSortStable, Root, Container } from '@ni2khanna/uikit'
import { forwardHtmlEvents } from '@pmndrs/pointer-events'
```

Now, we can start defining the actual layout. Every layout must start with a `Root` element (or an element that wraps the `Root` element, such as the `Fullscreen` component). In this example, the `Root` is of size 8 by 4 (three.js units). The `Root` has a horizontal (row) flex-direction, with 2 `Container` children, filling its width equally with a margin around them.

More in-depth information on the Flexbox properties can be found [here](https://yogalayout.dev/docs/).

```ts showLineNumbers
const canvas = document.getElementById('root') as HTMLCanvasElement
const renderer = new WebGLRenderer({ antialias: true, canvas })
renderer.localClippingEnabled = true
renderer.setTransparentSort(reversePainterSortStable)

const camera = new PerspectiveCamera(70, 1, 0.01, 100)
camera.position.z = 10
const scene = new Scene()

const { update } = forwardHtmlEvents(canvas, camera, scene)

const root = new Root(camera, renderer, {
  backgroundColor: 'red',
  sizeX: 8,
  sizeY: 4,
  flexDirection: 'row',
})
scene.add(root)

root.add(new Container({ flexGrow: 1, margin: 32, backgroundColor: 'green' }))
root.add(new Container({ flexGrow: 1, margin: 32, backgroundColor: 'blue' }))
```

All properties of the user interface elements can be modified using `container.setProperties({...})`. The last step is to set up the frame loop and handle resizing. Notice that the root component needs to be updated every frame using `root.update(delta)`.

```ts showLineNumbers
function updateSize() {
  renderer.setSize(window.innerWidth, window.innerHeight)
  camera.aspect = window.innerWidth / window.innerHeight
  camera.updateProjectionMatrix()
}
updateSize()
window.addEventListener('resize', updateSize)

let prev: number | undefined
renderer.setAnimationLoop((time: number) => {
  const delta = prev == null ? 0 : time - prev
  prev = time
  update()
  root.update(delta)
  renderer.render(scene, camera)
})
```

The final result shows a red root container with two children — one green and one blue — laid out horizontally with margins.
