title: Theming

When installing components from any kit, the `uikit cli` always adds a `theme.ts` file. The values inside this file can be freely adapted to create a custom theme.

For instance, when installing a `Button` from the `default` kit, the primary color of the theme can be modified as seen in the following.

```ts
import { basedOnPreferredColorScheme } from '@ni2khanna/uikit'

export const colors = basedOnPreferredColorScheme({
  light: {
    // ...
    primary: 'red',
    // ...
  },
  // ...
})
```

Now, with the following code

```ts
import { PerspectiveCamera, Scene, WebGLRenderer } from 'three'
import { reversePainterSortStable, Root, Text, setPreferredColorScheme } from '@ni2khanna/uikit'
import { Button, Defaults } from '@ni2khanna/uikit-default'

setPreferredColorScheme('light')

const canvas = document.getElementById('root') as HTMLCanvasElement
const renderer = new WebGLRenderer({ antialias: true, canvas })
renderer.localClippingEnabled = true
renderer.setTransparentSort(reversePainterSortStable)

const camera = new PerspectiveCamera(70, 1, 0.01, 100)
camera.position.z = 0.5
const scene = new Scene()

const root = new Root(camera, renderer, {})
scene.add(root)

const button = new Button({})
const text = new Text({ text: 'I am red.' })
button.add(text)
root.add(button)
```

the button's primary color will be red based on the theme configuration.
